# Low-level C Code Examples for the Raspberry Pi Pico 2W

This repository contains code for the formative coursework of **COMP2323: Computer Systems II** at the **University of Southampton**.  
Most of the code is adapted from the official **Pico SDK examples**.

> Coursework is ongoing

## Projects / Examples

- **usbIO** – Simple input/output via USB for a TTY terminal and the Pico 2W  
- **HttpClient** – HTTP client making requests to an API to fetch sunrise and sunset times in London  
- **NtpClient** – NTP client fetching the current time from a server  
- **http-server-example** – Web server running on the Pico, hosting a website that:
  - Allows toggling the onboard LED  
  - Displays the current temperature of the Pico

## Tech Stack
- **Hardware:** Raspberry Pi Pico 2W  
- **Language:** C  
- **SDK:** Raspberry Pi Pico SDK  

## Notes
- All examples are **adapted from the Pico SDK** and coursework instructions  
- Focused on **low-level embedded systems programming** and microcontroller fundamentals
